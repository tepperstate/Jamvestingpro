# CLAUDE.md (investors-p2bmarkets)

A modern, high-tech crypto/stock investment platform built with Laravel 10+, Tailwind CSS (or Bootstrap 5), and Glassmorphism design aesthetics.

## Project Structure

```
app/              # Core application code (Models, Http Controllers, etc.)
  ├── Http/       # Controllers, Middleware, Requests
  └── Models/     # Eloquent models
database/         # Migrations, seeders, and factories
public/           # Publicly accessible assets (images, CSS, JS)
resources/        # Frontend assets and Blade templates
  └── views/      # Blade templates (e.g. user dashboard, admin panel, exchange)
routes/           # Route definitions (web.php, api.php)
tests/            # PHPUnit/Pest automated tests
```

## Organization Rules

**Keep code organized and modularized:**
- API routes → `routes/api.php`, Web routes → `routes/web.php`
- Views → `resources/views`, organized by component/domain (e.g., `admin/`, `user/`)
- Utilities/Services → Extract heavy logic from controllers into dedicated Service classes
- Design → Maintain the high-tech, glassmorphism, dark-mode styling conventions using animejs for micro-animations.

**Modularity principles:**
- Single responsibility per controller/service
- Keep views clean; use Blade components for reusable UI elements
- Maintain the premium, dynamic aesthetic across all new pages (vibrant colors, glass panels, smooth hover effects).

## Code Quality - Zero Tolerance

After editing ANY file, run:

```bash
php artisan optimize:clear
php artisan view:clear
```
(If frontend assets are modified, ensure Vite/mix is built if required)

Fix ALL errors/warnings before continuing.
